# ASP.NET Core MVC Mastery

> A personal learning journey to master ASP.NET Core MVC through real-world projects, from fundamentals to enterprise-level architecture.

## 📌 About This Repository

This repository is my personal reference and learning journey for mastering **ASP.NET Core MVC**.

The goal is not just to build projects, but to understand how professional .NET applications are designed, implemented, reviewed, and improved.

Every project introduces new concepts while reinforcing previous ones using better architecture and cleaner code.

# 🧠 Learning Philosophy

This repository follows one simple rule:

> **Understand first. Code second.**

Every feature will be implemented only after understanding:

* Why it exists
* How it works internally
* When to use it
* Best practices
* Common mistakes

---

# 📖 Learning Workflow

For every Sprint:

1. Learn the concept
2. Design the solution
3. Build the feature
4. Review the code
5. Refactor if needed
6. Answer interview questions
7. Commit changes
8. Update documentation
