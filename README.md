# 📖 About

This repository documents my complete journey of learning **ASP.NET Core MVC**.

The objective is not only to build projects, but to deeply understand how ASP.NET Core MVC works internally, write clean and maintainable code, and gradually learn the practices used in professional software development.

Every project introduces new concepts while reinforcing previous ones using better architecture and cleaner implementation.

This repository also serves as my personal reference for future revision.

# 🚀 Learning Philosophy

This journey follows one simple principle:

> **Understand First. Code Second.**

Every concept will be learned by answering these questions:

* Why do we need it?
* How does it work internally?
* When should we use it?
* What are the best practices?
* What are the common mistakes?

Only after understanding the concept will we implement it.
