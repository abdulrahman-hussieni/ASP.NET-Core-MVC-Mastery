# 🗺 Project 01 Roadmap

> **Project:** Library Management System
> **Learning Goal:** Master ASP.NET Core MVC by understanding every layer of the framework while building a real application.

---

# 🎯 Project Philosophy

This project is **not** about building a Library Management System only.

The real goal is to understand **how ASP.NET Core MVC works internally**.

Every Sprint answers one question.

Every concept is learned before it is used.

Every feature is built only after understanding the technology behind it.

---
